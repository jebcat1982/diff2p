diff2p
======

`diff2p` is a two pane diff for console (terminal).

Availability
------------

Available for Linux, BSD, OSX or Cygwin.

Requirments
-----------

A coloured terminal. Python 2.x (tested with 2.7).

Usage
-----

`diff2p file-one file-two`
